Traditional flow of HTML elemtns are to be stacked on top of each other. These types of elements are knowon as block elements

Inline elements are those that flow with the text and do not start on a new line. A good example is the span tag. These elements do not maintain their size and shape.

Inline-block elements are those that flow with the text and also maintain their size and shape.
